AICommand

A natural language interface for executing terminal commands

Installation

Install AICommand with just one command and start using natural language to interact with your terminal.

bash -c "$(curl -fsSL https://ugoi.github.io/AICommand_Page/download/install.sh)"

Uninstallation

Uninstall AICommand with just one command and remove it from your system.

bash -c "$(curl -fsSL https://ugoi.github.io/AICommand_Page/download/uninstall.sh)"

Usage

Once installed, you can start using AICommand by running:

ai_command <your-natural-language-command>

Here are a few examples of natural language commands you can use:

  • Create a new folder named MyProject
  • Show me the list of files in the current directory
  • Delete the file called example.txt
